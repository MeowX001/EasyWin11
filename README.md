# EasyWin11
Simple CLI tool to optimize your Windows experience

version 1.0.1 notes  
The flush ram function now directly checks if the user has RAMMAP installed or not  

Notes for powerusers:
First run — RAMMap EULA: EasyWin11 automatically accepts the RAMMap EULA via registry on first use. If for any reason it can't, it will open RAMMap and ask you to click Agree once manually, then the feature will work normally from that point on.  
Antivirus false positives: Windows Defender may flag the .exe build or the RAMMap download. Add your EasyWin11 folder to Defender exclusions if you hit issues during PyInstaller builds or on first launch.
